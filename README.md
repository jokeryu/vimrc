# vimrc
my vimrc

## Requirements
### VIM
``` sh
git clone https://github.com/gmarik/Vundle.vim ~/.vim/bundle/
```
``` vim
:PluginInstall
```

### System
``` sh
sudo pacman -S ctags --needed
```

### resources
``` sh
cp ~/.vim/fonts/SourceCodePro-Regular-Powerline.otf ~/.fonts/
```

### Front-end
``` sh
sudo npm install autoprefixer -g
gem install compass sass
```
